### Adicionar repositorio ao dbeaver

```bash
sudo add-apt-repository ppa:serge-rider/dbeaver-ce
```

### Atualizar pacortes

```bash
sudo apt-get update
```
### Instalando dbeaver

```bash
sudo apt-get install dbeaver-ce
```
